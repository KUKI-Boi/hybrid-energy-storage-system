## 📌 Overview
This project presents an **interactive Hybrid Energy Storage System (HESS) simulator** developed as a **web-based application**. The system demonstrates how multiple energy storage technologies can work together to efficiently supply power to a load.

Hybrid Energy Storage Systems combine two or more storage technologies (such as **batteries and supercapacitors**) to improve system performance, energy efficiency, and power management compared to using a single storage device. :contentReference[oaicite:0]{index=0}

The website provides a **visual and interactive platform** to understand how hybrid energy storage systems manage power flow and respond to changing energy demands.

## ⚙️ System Concept

A Hybrid Energy Storage System typically integrates:

- **Battery** → Provides long-term energy storage  
- **Supercapacitor** → Handles fast power fluctuations  
- **Energy Management System (EMS)** → Controls power distribution between devices  

```
Power Source
      │
      ▼
Energy Management Controller
      │
 ┌───────────────┬───────────────┐
 ▼               ▼
Battery       Supercapacitor
(Energy)        (Power)
 └───────────────┬───────────────┘
                 ▼
               Load
```

This combination allows the system to **handle both steady energy supply and sudden power demands efficiently**. :contentReference[oaicite:1]{index=1}
